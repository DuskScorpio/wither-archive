## Description
- An intermediate 12gt 7 type tree farm.
## Positives
- All trees work with the same sequence, no mode switching needed
- Very compact size, TNT version fits within single chunk
## Negatives
- Lower rates compared to other related designs
- Hopper cart spam in the main pusher, can be replaced by adding a wall behind the main pusher and hoppers to collecting the saplings glitching out
## Design Specifications
- Dimensions:
  - Core: 9x19x9
  - TNT duping variant: 9x19x16
  - Wither variant: 9x19x20
### Notes
- For oak, place immovable block 8 blocks above the dirt (7 blocks gap between sapling and the height limiting block) and remove the stripped log in main pusher at the same height.
- See Figure 2 for a comparison to other related designs.
### Build
- The design is directional. Do not rotate.
- Fill up the bone meal hoppers and hopper carts with bone meal before building the bone meal unloader.
- Set up the stick filter as shown in Figure 3.
- See Figure 4 for a demonstration of hopper minecart alignment.
### How to use
1. Flick lever. 
2. Open spruce trapdoor and stand on top of the chain
3. Close the trapdoor to prevent player sneaking when holding shift
4. Walk backwards towards the fence gate and to the left
5. Hold shift and use continuously against the dispenser face
