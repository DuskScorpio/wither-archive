### How to use
1. Place a stack of propagule into the player's main hand.
2. Open the trapdoor above the stone brick slab and stand on redstone block while sneaking.
  - When using a carpet bot, use the following command: `/player name spawn at X+0.05 Y+1 Z+0.05` (X,Y, and Z denote the respective coordinates of the stone brick slab).
3. Move forward to align against the two stone brick walls (Refer to Figure 1). 
4. Hold down sneak, then close the trapdoor.
5. Flick the lever to toggle the farm.
6. Aim at the dispenser using zoom or a spyglass, making sure the player can plant on the dirt (refer to Figure 2).
  - When using a carpet bot, use: `/player name look 3.6 103.7`.
7. Right click with 1gt interval to plant the propagules.
