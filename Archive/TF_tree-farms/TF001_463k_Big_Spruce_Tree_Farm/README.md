## Description
- An advanced 12gt big spruce tree farm.
## Positives
- TNT variant: 99.855% efficiency
- Compact (14x35x14)
### Notes
- This design is directional. Do not rotate.
- See [this video](<https://youtu.be/Mb0VzR_-Mfo>) for further instructions.
### Build
- (1.20.2+) Use 3 snow layers instead of 4 for llama boat alignment.
- Add a roof over the farm if building in the Overworld.
- Ensure the light level of all podzol is lower than 9.
### How to use
- (1.17-1.20.6) Refer to [this video](<https://youtu.be/Mb0VzR_-Mfo?&t=221>) for instructions on how to use the farm.
- (1.21+)
  1. Flick the lever to toggle the farm.
  2. Use the trapdoor shown in Figure 1 to go into crawl mode and mount the pig.
  3. Look away from dispenser (refer to Figure 2 for player rotation).
  4. Use continuously or at a 1gt (0.05 second) interval to plant saplings.
