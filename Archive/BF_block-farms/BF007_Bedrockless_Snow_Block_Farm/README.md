**Endorsed by:** *Scorpio*

**Tags:** *Tested & Functional*

**Original post:** [View on Discord](https://discord.com/channels/913065809096638494/1497625497566314787)

Snow farm that uses a wither to break snow layers and crafts them into snow blocks.
## Versions
1.21.5+
## Rates
- Snow Block: 7200/h
## Build
- Make sure that the entire farm is inside 1 chunk

Follow the schematics in this order: 
1  - Summoning
2 - Snow Golems Alignment
3 - Bedrockless Wither Snow Block Farm
## Considerations
- Only works in the overworld
## Notes
- Do not build in biomes with a base temperature greater than 0. Add a roof if you're building it in a biome in which it can rain
- Nametag the creaking
